Java programavimo sistemoje programuotojo paraðytas kodas kompiliuojamas ne á procesoriui specifinæ, o á tarpinæ formà. Java ðiukðliø surinkëjas pats nustato, jog struktûra nebenaudojama ir jos uþimama atmintis gali bûti panaudota kam kitam.

Ávairios Java programos dalys nesunkiai gali bûti vykdomos lygiagreèiai (multithreading).

Java taip pat turi gerai iðvystytà refleksijos sistemà.

Pradedant naujesne 1.5 versija, java kalba taip pat palaiko ¥bendrybes´ (angl. generics), kurios primena aiðkesná, paprastesná praeityje kai kuriose kalbose buvusiø makrokomandø variantà.
